- Drawer sticks or drags
- Drawer tilts to one side
- Grinding noise when opening/closing
- Drawer jumps off track

## Print Settings

### Wheel (W11578839) - PETG or PLA
| Setting | Value |
|---------|-------|
| Layer Height | 0.15-0.2mm |
| Walls | 3-4 |
| Infill | 100% |
| Supports | No |
| Orientation | Flat on bed |

### Axle (W11573594) - TPU
| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |
| Walls | 4 |
| Infill | 100% |
| Supports | No |
| Speed | 25-30mm/s |
| Orientation | Shoulder down |

> **Important:** The axle MUST be printed in TPU for the snap-fit legs to flex without breaking.

## Assembly

1. Print wheel in PETG or PLA (rigid)
2. Print axle in TPU (flexible)
3. Insert axle legs through wheel's 9mm center hole
4. TPU legs compress to fit, then snap to retain wheel
5. Install assembly in refrigerator drawer track

## Compatibility

- Whirlpool refrigerators
- KitchenAid refrigerators
- Maytag refrigerators
- Kenmore (Whirlpool-made) refrigerators
