## Tips for Appliance Parts

### Material Selection

| Application | Recommended | Avoid |
|-------------|-------------|-------|
| Seals/Gaskets | PETG, TPU | PLA |
| Knobs/Handles | PETG, ABS | - |
| Structural | PETG, ABS | PLA (brittle) |
| High Temp (>60°C) | ABS, ASA | PLA, PETG |
| Food Contact | PETG | ABS |

### General Guidelines

1. **Measure twice** - Use calipers for accuracy
2. **Test fit first** - Print a thin slice to verify dimensions
3. **Check tolerances** - Add 0.2-0.3mm clearance for fits
4. **Consider shrinkage** - Some materials shrink more than others
