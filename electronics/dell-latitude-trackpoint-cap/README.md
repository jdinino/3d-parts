## Compatible Laptop Models

### Dell
- Latitude E7440
- Other Dell Latitude models with same TrackPoint stem (square 3.1mm)

## Specifications

| Parameter | Value |
|-----------|-------|
| **Total Height** | 3.75 mm |
| **Outer Diameter** | 8.65 mm |
| **Inner Diameter** | 7.65 mm |
| **Square Hole** | 3.1 x 3.1 mm |
| **Hole Depth** | 1.9 mm |

## Installation

1. Remove old or damaged TrackPoint cap from keyboard stem
2. Press new cap onto square stem until seated
