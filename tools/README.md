## Tips for Tool Parts

### Material Selection

| Application | Recommended | Notes |
|-------------|-------------|-------|
| Jigs/Fixtures | PETG, ABS | Dimensional stability |
| Handles | TPU, PETG | Grip and comfort |
| Organizers | PLA, PETG | Low stress |
| Measuring tools | PETG | Stability and accuracy |
| Soft jaws | TPU | Protect workpieces |

### Considerations

- Dimensional accuracy for jigs
- Wear resistance for frequently used tools
- Ergonomics for handles
- Consider printing solid (100% infill) for strength
